# 第 79 节：Remote State Data Source 实操

## 知识点讲解

本节来自 doc/79.txt。核心主题是：完整实现 network 与 security 两个目录，security 通过 terraform_remote_state 自动读取 network 输出并生成规则说明。

	erraform_remote_state 是 Terraform 内置数据源，用来读取另一个 Terraform state 文件中声明为 output 的值。课程中的典型场景是 networking team 创建公网 IP，并把 state 放在 S3；security team 不手工复制 IP，而是从 remote state 自动读取，再用于防火墙规则。

## 实验目标

1. 创建 
etwork 配置，并把 state 写入 LocalStack S3 backend。
2. 在 consumer 配置中使用 data "terraform_remote_state" 读取 network 输出。
3. 验证 consumer 的 plan/apply 能自动使用 network 输出。
4. 按正确顺序销毁 consumer、network，并清理 LocalStack 中的 state 对象。
