# 第 140 节：Auto Scaling Group 与 Launch Template

## 知识点讲解

本节对应 `doc/140.txt` 的课程录音转写。主题是 `Auto Scaling Group 与 Launch Template`，重点不是把真实 AWS 账号作为练习环境，而是在本地 LocalStack 中理解 Terraform AWS Provider 如何声明、查询、绑定和销毁 AWS 资源。

LocalStack 使用 `http://localhost:4566` 作为 AWS API endpoint，凭证固定为 `test/test`。这可以训练 Terraform 写法、依赖关系、状态变化和输出设计，但不能替代真实 AWS 的 IAM 权限评估、真实扩缩容或真实网络连通性。

## 实验目标

- 创建 launch template。
- 建模使用 launch template 的 Auto Scaling Group 关键配置。
- 理解 LocalStack Community 不支持真实创建 ASG，本实验只验证 Launch Template 与 ASG 配置模型。

## 实验边界

- 不访问真实 AWS 账号。
- 不写入真实 access key、secret key 或 root 账号信息。
- 使用 Terraform AWS Provider + LocalStack 进行练习。
- 本节包含 LocalStack Community 无法完全复刻的真实 AWS 行为。LocalStack Community 对 Auto Scaling Group 的 CreateAutoScalingGroup API 返回 501，因此参考实现只真实创建 Launch Template，并用 local.asg_spec 建模 ASG 关键字段；真实 ASG 扩缩容需要在真实 AWS 或支持该 API 的环境中验证。
